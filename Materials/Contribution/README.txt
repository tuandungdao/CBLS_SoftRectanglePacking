Hướng dẫn cài đặt:
1. Import thư mục "Mã nguồn" và import thư viện chương trình Java trên các IDE hỗ trợ ngôn ngữ lập trình Java (Eclipse, Netbeans, ...) 
2. Sửa địa chỉ đường dẫn đọc file dữ liệu, địa chỉ đường dẫn thư mục "Instances_Soft_Packing", 
sửa đường dẫn địa chỉ ghi file kết quả (.txt và .xlsx) hoặc comment phần này để kết quả hiển thị ra màn hình console
3. Thực hiện chạy chương trình
